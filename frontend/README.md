# ✨ Javino Frontend: Premium Forensic Interface

**Proprietary Software by [Sriram S](https://sriram.website/)**

The **Javino Frontend** is a high-end, production-grade interface designed for the visual analysis of media authenticity. It prioritizes clarity, professional aesthetics, and smooth interactivity to make forensic data accessible.

## 🎨 Design Philosophy

- **Apple-Inspired Aesthetic**: Minimalist, clean, and focus-driven.
- **Dynamic Interactions**: Leveraging **Framer Motion** for physics-based transitions and data visualization.
- **Micro-Animations**: Subtle feedback for forensic signal changes.
- **Responsive Forensics**: Tailored layouts for both mobile field-checks and desktop laboratory analysis.

## 🛠️ Technical Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typeface**: Geist (Sans & Mono)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation
1.  Navigate to the directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## 🏗️ Key Components

- **Analysis Dashboard**: Real-time visualization of forensic probability scores.
- **Signal Feed**: Sequential breakdown of EXIF and Vision flags.
- **Forensic Report Viewer**: Authority-formatted display of generated LLM reports.
- **Media Uploader**: Drag-and-drop interface with instant preview and size optimization.
